## Detailed Description

NightstandPlus B2B Product Finder is a lightweight Chrome extension for furniture importers, wholesalers, hotel project buyers and sourcing teams who need faster access to verified NightstandPlus product and OEM/ODM resources.

The extension works without relying on a website search function. It keeps a local product index and opens verified official NightstandPlus pages only when the user clicks a link.

Core features:

- Search nightstand, bedside table and bedside cabinet product references by size, feature, category, market or sourcing need.
- Open stable official NightstandPlus pages for products, smart nightstands, wireless charging, LED, safe storage, custom products, OEM/ODM and contact.
- Compare two product profiles by dimensions, buyer features, MOQ, packaging and official URL.
- Save favorite product references locally in Chrome.
- Export product results as CSV or JSON for internal sourcing records.
- Generate an English RFQ inquiry draft for OEM/ODM and wholesale communication.

This extension is built for B2B users who research nightstand OEM, bedside table wholesale, bedside cabinet factory supply, smart nightstands, bedroom furniture sourcing and hotel furniture projects.

## Permissions Justification

Permission: `storage`

Justification:

The extension uses Chrome local storage only to save the user's favorite product references on their own browser. This data is not transmitted to NightstandPlus or any third-party service.

## Data Usage

The extension does not collect, transmit, sell or share personal data.

The extension does not read browsing history, page content, tabs, cookies, forms, account data or user activity on other websites.

CSV and JSON exports are generated locally in the browser from product data shown inside the extension.
